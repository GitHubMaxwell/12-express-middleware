CORS Demo

This file brings in jQuery and then tries to hit your restful API. If you have installed the cors middleware in your server and gave it no parameters, then in your browser console, you should see data from your API. If you give it a parameter that sets the origin to some domain that is not the same as the one you've installed this index.html file on ... you should get an Access Control error in your console.

Welcome to CORS.

  • Edit this file, changing the URL in the script tag below to point to the public IP/Port for your API Server
  • Upload this to a live web server, like ghpages, or your own domain.
